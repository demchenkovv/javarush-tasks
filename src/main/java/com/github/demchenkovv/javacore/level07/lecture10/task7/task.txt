Когда следует применять синхронизацию? Используйте правило синхронизации Брайана*:
«Если вы записываете данные в переменную, которая может быть затем прочитана другим потоком,
или читаете данные из переменной, которая могла быть записана другим потоком, вы должны использовать синхронизацию.
Кроме того, и читающий, и записывающий потоки должны синхронизироваться по одной блокировке».
*От Брайана Гетца, соавтора книги «Java Concurrency in Practice» (рус: "Java Concurrency на практике").
