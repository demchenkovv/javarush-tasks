Объектный граф при сериализации Java

Основные тезисы:

- Нам не обязательно явно сериализовывать ссылочные объекты.

- Объектный граф - это набор объектов, которые будут сериализованы автоматически, если объект, содержащий
  ссылку на них, сериализован. Другими словами, мы можем сказать, что когда мы сериализуем любой объект и если
  он содержит любую другую ссылку на объект, то JVM сериализует объект, а также ссылки на его объекты.

- в объектном графе каждый объект должен быть сериализуемым. Если хотя бы один объект не является сериализуемым,
  то мы получим выполнение во время выполнения со словами NonSerializableExecution.
