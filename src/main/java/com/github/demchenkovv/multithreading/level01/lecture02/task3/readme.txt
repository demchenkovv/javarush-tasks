Моя логика, если кому поможет.

Со школы ещё знаю, что AND (или И или &&)  это логическое умножение, а OR  (или ИЛИ или ||) это логическое сложение.
Следуя этой логике получаем:

1. (a && b && c && !d) || (!a && c) || (!b && c) || (c && d)
2. (a * b * c * !d) + (!a * c) + (!b * c) + (c * d)

Убираем скобки так как умножение в любом случае производиться в первую очередь.

3. a * b * c * !d + !a * c + !b * c + c * d

Выносим общий множитель за скобки, это c.

4. c * (a * b * !d + !a + !b + d)

Берём логику двоичной системы, что true = 1, а false = 0 получаем:

5. c * (1 * 1 * 0 + 0 + 0 + 1)
6. c * (0 + 0 + 0 +1)
7. c * 1

Возвращаем логические символы true = 1 и && = * получаем:

8. с && true

true всегда true и от этого ничего не зависит и в итоге получаем, что всё зависит только от c получаем:

9. return с
